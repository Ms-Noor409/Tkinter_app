# Contact Form Application

## Overview

A simple Contact Form Application built with Python Tkinter. The application allows users to enter their name, contact number, and email address. It validates the input data and stores the submitted information in a text file.

## Features

* User-friendly GUI built with Tkinter
* Name validation (letters only)
* Contact number validation (11 digits required)
* Email validation using Regular Expressions
* Data storage in a local text file (data.txt)
* Success and error message popups

## Technologies Used

* Python
* Tkinter
* Regular Expressions (re)

## How It Works

1. Enter your name.
2. Enter an 11-digit contact number.
3. Enter a valid email address.
4. Click the Submit button.
5. The data is validated and saved to data.txt.
